# Amazon E-Commerce Sales Analysis | Python EDA + Power BI Dashboard

## 📊 Project Overview

This project performs Exploratory Data Analysis (EDA) on Amazon e-commerce sales data using Python and builds a professional interactive Power BI dashboard to analyze revenue, order volume, delivery performance, and customer purchasing behavior.

The objective is to clean raw data, extract business insights, and visualize key performance metrics for decision-making.

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI
* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Visualization

---

## 🔍 Python EDA Tasks Performed

* Cleaned dataset by handling missing values and correcting data types
* Performed exploratory data analysis on sales, orders, and categories
* Created new features like weekend indicator and delivery flags
* Analyzed category-wise sales performance
* Examined courier status and return trends
* Identified peak sales periods and customer behavior

---

## 📈 Power BI Dashboard Features

* KPI Cards (Total Revenue, Orders, Avg Order Value, Return Rate)
* Revenue and Orders trend analysis
* Category performance comparison
* Delivery status distribution
* Sales channel contribution
* State-wise order analysis
* Interactive slicers for dynamic filtering

---

## 💡 Key Insights

* Certain product categories contribute majority of revenue
* Online sales channel dominates order volume
* Peak sales observed during specific time periods
* Delivery success rate significantly higher than returns
* Few states generate majority of orders
* Shipping speed impacts customer order volume
